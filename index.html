บทเรียนปฏิบัติการ: สร้างระบบประเมินสุขภาพจิต THI-15

เรียนรู้การจัดเก็บข้อมูล, การใช้ AI ช่วยเขียนโค้ด และการขึ้นระบบด้วย Google Apps Script

1. เริ่มต้นพื้นที่จัดเก็บด้วย Google Drive

ก่อนเริ่มต้นสร้างไฟล์ฐานข้อมูล สิ่งแรกที่ต้องทำคือการจัดเตรียมโครงสร้างโฟลเดอร์ให้เรียบร้อย

ทำไมต้องสร้าง "โฟลเดอร์ใหม่" ทุกครั้งก่อนเริ่มชิ้นงาน?
  • ความเป็นระเบียบและปลอดภัย: ป้องกันไม่ให้ไฟล์ระบบปะปนกับไฟล์ส่วนตัวอื่นๆ บน Drive
  • การจัดการสิทธิ์ (Permission): หากต้องการแชร์ระบบให้ทีมงานร่วมพัฒนา การแชร์ทั้งโฟลเดอร์จะทำให้ไฟล์ย่อยและหน้าเว็บได้รับสิทธิ์เข้าถึงที่ถูกต้องทันที ไม่ต้องตามแก้ทีละไฟล์
  • ง่ายต่อการเคลื่อนย้ายและสำรองข้อมูล: เมื่อระบบเสร็จสมบูรณ์ สามารถดาวน์โหลดหรือย้ายโครงการนี้ไปไว้ที่อื่นได้ในคลิกเดียว

วิธีทำ: เข้าไปที่ Google Drive ของท่าน -> คลิกปุ่ม + ใหม่ (+ New) -> เลือก โฟลเดอร์ใหม่ -> ตั้งชื่อว่า "ระบบประเมิน THI-15"

2. การออกแบบฐานข้อมูล (Google Sheets 3 ชีท)

ดับเบิ้ลคลิกเข้าโฟลเดอร์ที่สร้างไว้ จากนั้นสร้าง Google Sheets ขึ้นมา 1 ไฟล์ และตั้งชื่อชีทย่อยด้านล่างให้ครบ 3 ชีทเพื่อทำหน้าที่เป็นตารางเก็บข้อมูล:

ชีทที่ 1: member (ตารางเก็บข้อมูลสมาชิก)

st_id (Primary Key) name DOB id_course

ชีทที่ 2: course (ตารางตัวเลือกหลักสูตร)

id_course course

ชีทที่ 3: THI-15 (ตารางบันทึกผลการประเมิน)

no date_time st_id q1 q2 ... q15 total result

3. เทคนิคการใช้ AI ช่วยเขียนโค้ด (Prompt Engineering)

ในการเขียนระบบ Web App ขนาดใหญ่ เราสามารถใช้ AI สั่งงานเขียนโค้ดผ่านข้อความคำสั่ง (Prompt) ที่มีความชัดเจน ดังตัวอย่างด้านล่างนี้:

COPY PROMPT นี้ไปสั่งงาน AI ได้เลย
1.จงเขียนโค้ด Google Apps Script (ไฟล์ Code.gs และ Index.html ร่วมกันแบบ Single Page Application) เพื่อสร้างเว็บแอปประเมินสุขภาพจิต THI-15 2.โดยเชื่อมต่อกับ Google Sheets ที่มี 3 ชีทคือ member, course, และ THI-15 ข้อกำหนดระบบ: 1. หน้าสมัครสมาชิก: ใช้ชีท 'member' ส่วน DOB ให้กรอกเป็นตัวเลยเรียง เช่น "29102534" ให้ดึงข้อมูลตัวเลือกวิชามาจากชีท 'course' และเก็บข้อมูลลงชีท 'member' โดยใช้ฟิลด์ st_id เป็นคีย์หลัก ป้องกันไม่ให้สมัครซ้ำ จำเป็นต้องกรอก 2. หน้า Login: ล็อกอินโดยด้วย st_id และรหัสผ่านด้วย DOB จากชีท 'member' และมีระบบ Logout 3. หน้าฟอร์มประเมิน THI-15: ให้แสดงคำถาม 15 ข้อ สร้างเป็นตารางกริดหลายตัวเลือก ข้อคำถาม หัวตารางเป็น รายการประเมิน ส่วนหัวตารางตัวเลือก ประกอบด้วย ไม่เลย เล็กน้อย มาก มากที่สุด ส่วนการเก็บข้อมูลลงตารางให้เป็นค่าตัวเลขตามการให้คะแนนรายข้อ ข้อคำถาม q1 ท่านรู้สึกพึงพอใจในชีวิต q2 ท่านรู้สึกสบายใจ q3 ท่านรู้สึกเบื่อหน่ายท้อแท้กับการดำเนินชีวิตประจ าวัน q4 ท่านรู้สึกผิดหวังในตัวเอง q5 ท่านรู้สึกว่าชีวิตของท่านมีแต่ความทุกข์ q6 โดยแสดงผลประเมินครั้งล่าสุดอยู่ด้านบน ดึงเฉพาะข้อมูลของคนที่ล็อกอินอยู่เท่านั้น และอนุญาตให้ทำประเมินซ้ำได้ 6. ขอผลลัพธ์ในโทนสีฟ้า และจัดรูปแบบให้เข้าใจง่าย

4. วิธีนำโค้ดไปใส่ใน Apps Script และเปิดใช้งาน

เมื่อได้โค้ดจาก AI มาแล้ว ให้ทำตามขั้นตอนแบบละเอียดดังนี้เพื่อสร้างระบบจริงให้ใช้งานได้:

  1. เปิดหน้าต่าง Apps Script: ที่หน้า Google Sheets ของคุณ ให้ไปที่เมนูด้านบน เลือก ส่วนขยาย (Extensions) -> Apps Script
  2. วางโค้ดฝั่งเซิร์ฟเวอร์:
    • เปิดไปที่ไฟล์ชื่อ รหัส.gs (Code.gs)
    • ลบโค้ดเก่าออกให้หมด แล้วนำโค้ดที่ได้จาก AI ในส่วนของ Google Apps Script มาวางแทนที่
    • *อย่าลืม* นำ ID ของ Google Sheets มาใส่ในตัวแปรเชื่อมต่อข้อมูลให้ถูกต้อง
  3. สร้างไฟล์หน้าเว็บ (HTML):
    • คลิกที่เครื่องหมาย + ข้างคำว่า "ไฟล์" (Files) ในแถบซ้ายมือ
    • เลือก HTML แล้วตั้งชื่อไฟล์ว่า Index
    • ลบโค้ดตั้งต้นออก นำโค้ดหน้าตาเว็บที่เป็นส่วนของ HTML/CSS/JavaScript ที่ AI เจนให้มาใส่ทั้งหมด
  4. บันทึกโครงการ: กดปุ่มไอคอนแผ่นดิสก์ หรือกด Ctrl + S เพื่อบันทึก
  5. การเปิดใช้งานแอป (Deployment):
    • คลิกปุ่มสีน้ำเงินขวาบนคำว่า การทำให้ใช้งานได้ (Deploy) -> เลือก การทำให้ใช้งานได้ใหม่ (New deployment)
    • คลิกไอคอนฟันเฟือง เลือกประเภทเป็น เว็บแอป (Web app)
    • ตั้งค่าตามนี้:
      - เรียกใช้ในฐานะ: ฉัน (อีเมลของคุณ)
      - ผู้มีสิทธิ์เข้าถึง: ทุกคน (Anyone)
    • กดปุ่ม การทำให้ใช้งานได้ (Deploy)
    • หากมีหน้าต่างถามหาความปลอดภัย ให้กด ให้สิทธิ์เข้าถึง (Authorize access) เลือกอีเมลของคุณ กด Advanced และเลือก Go to... (unsafe) เพื่ออนุญาตสิทธิ์ให้กับระบบดึงข้อมูลในชีท
  6. รับลิงก์ใช้งาน: ระบบจะมอบ URL สำหรับใช้งานเว็บแอปพลิเคชันมาให้ คุณสามารถนำลิงก์นี้ไปแชร์ให้นักศึกษาทำแบบประเมินได้ทันที!
